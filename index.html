Тема проекта


Видео игра шутер, с реализацией механик, не менее 30-ти

Описание разработанного продукта


Игра поддерживает одиночный и многопользовательский режимы. В многопользовательской игре игроки делятся на две команды, после чего сражаются друг с другом несколько раундов, по результатам которых определяется команда-победитель.

Чтобы выполнять задачи по стрельбе и уничтожению врагов, игроку дают различные средства: оружие, технику, спецсредства.

Игроку доступны разные режимы игры, меняющие условия поединка между командами, а также игра с ботами.

Алмазы - уровневая валюта игры, вы получаете определенное количество алмазов за каждый уровень игрока, их вы можете потратить на приобретение и кастомизацию персонажей.

Монеты - стандартная валюта игры, за нее вы можете повышать уровни ваших персонажей и покупать дополнительное снаряжение.

Уровень игрока - отображает общий прогресс игрока, вне зависимости от стороны, за которую вы играете.

Руководство пользователя

  1. Термины и сокращения
    Игра — шутер
    Исполняемый файл — файл, который может быть обработан или выполнен компьютером без предварительной трансляции
    Интерфейс — совокупность средств, методов и правил взаимодействия между элементами игры
    Курсор — элемент графического интерфейса, который указывает на объект, с которым будет производиться взаимодействие
    ПК — персональный компьютер
    Пользователь — зарегистрированный и авторизованный пользователь игры
    Windows — операционная система производства Microsoft
  2. Введение
    Шутер - многопользовательская компьютерная игра в жанре «шутер от первого лица», разработанная для ПК на операционной системе Windows. Игроки делятся на две команды, после чего сражаются друг с другом несколько раундов, по результатам которых определяется команда-победитель.
  3. Назначение шутера
    Чтобы выполнять задачи по стрельбе и уничтожению врагов, игроку дают различные средства: оружие, технику, спецсредства. Обычно существует три режима — командный, когда игроки объединяются в группы и сражаются друг с другом, кооперативный, когда игроки играют за одну команду или проходят игру вместе, и «все против всех», когда каждый сражается за себя. Частным случаем последнего является «дуэльный» режим, когда два игрока бьются друг против друга один на один.
  4. Предварительная подготовка
    1. Регистрация
      Регистрация нового пользователя осуществляется на сайте игры. Для регистрации нужно заполнить поля «Логин», «Пароль» и «Подтверждение пароля», после чего нажать кнопку «Зарегистрироваться».
    2. Установка клиента игры
      Для установки клиента, его предварительно нужно загрузить с сайта игры, для чего необходимо нажать на кнопку «Скачать». Клиент для Windows представляет собой в rar-архив. Распакуйте клиент средствами операционной системы или утилитами архивирования.
  5. Начало игры
    1. Запуск игры
      Для запуска игры используйте исполняемый файл shooter.exe, расположенный в папке игры.
    2. Авторизация
      После запуска игры пользователю потребуется провести процедуру авторизации используя логин и пароль, созданные при регистрации.
    3. Интерфейс игры
      Главное меню, открывающееся при запуске игры, содержит две кнопки: «Начать игру» и «Настройки». ПРи выборе начала игры, появляется окно с настройками будущей игры, где можно выбрать ее режим, игроков и дополнительные правила.

    Для работы программного продукта необходима следующая программно-аппаратная конфигурация:
    Windows 7, Windows 8, Windows 10;
    32-разрядные системы: компьютер, оборудованный процессором Intel или совместимым процессором с тактовой частотой 1 ГГц или выше;
